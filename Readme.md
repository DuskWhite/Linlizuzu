# 邻里租租
***
## 项目介绍
邻里租租是一个前后端分离的物品租赁系统,项目采用Springboot+vue开发.管理系统部分使用了Vue+element+admin模版
### 项目技术栈:
#### 后端技术栈
1. Springboot
2. WebScoket
3. Spring Security
4. Mybatis plus
5. Mysql
6. ...  
#### 前端技术栈
1. element-ui
2. Vue
3. axios
4. WebScoket
5. Vuex
6. ...

## 业务功能:
### 用户模块
1. 用户注册
2. 用户登录
3. 个人信息管理
4. 余额充值
   
### 物品模块
1. 物品发布与信息修改
2. 物品搜索
### 订单模块
1. 创建与修改订单
2. 订单查询
### 后台管理模块
1. 用户管理
2. 订单管理
3. 物品管理
### 聊天模块
